# zh.ru.net

Минималистичный личный навигационный хаб Алексея Жаркова. Главная страница объединяет три направления: Telegram-канал, личное портфолио и проект «Онлайн-портфолио».

Сайт состоит только из статических HTML, CSS и JavaScript-файлов. Сборка, установка зависимостей и серверная часть не требуются.

## Структура

- `index.html` — разметка страницы и SEO-метаданные.
- `styles.css` — оформление терминала и адаптивные стили.
- `script.js` — Canvas-анимация, расшифровка ссылок и управление клавиатурой.
- `favicon.svg` — иконка сайта.
- `robots.txt` и `sitemap.xml` — правила индексирования.
- `.htaccess` — редиректы, заголовки и кеширование для Apache.

## Локальный просмотр

Для быстрого просмотра можно открыть `index.html` непосредственно в браузере. Чтобы проверить сайт через локальный HTTP-сервер, выполните в каталоге проекта:

```powershell
python -m http.server 8000
```

Затем откройте `http://localhost:8000/`. Стандартный сервер Python не обрабатывает `.htaccess`, поэтому маршруты `/go/*` локально вернут 404 вместо редиректа.

## Размещение на Apache

Загрузите содержимое корня проекта, включая скрытый файл `.htaccess`, в корневой каталог домена `zh.ru.net`. На хостинге должны быть разрешены `.htaccess` и модуль `mod_rewrite`; для заголовков и сроков кеширования дополнительно используются `mod_headers` и `mod_expires` при их наличии.

Настроены временные редиректы `302`:

| Внутренний адрес | Назначение |
| --- | --- |
| `/go/tg` | `https://t.me/tolstuil` |
| `/go/portfolio` | `https://alexeyzharkov.ru/?utm_source=zh.ru.net&utm_medium=referral&utm_campaign=personal_hub` |
| `/go/product` | `https://сайт-портфолио.рф/?utm_source=zh.ru.net&utm_medium=referral&utm_campaign=personal_hub` |
